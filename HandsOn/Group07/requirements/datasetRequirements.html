Dataset Requirements

This document discusses whether each selected dataset satisfies the requirements. R1-R4 must be satisfied. R5-R6 are optional and their degree of satisfaction is discussed. Identifies the licenses of all the selected datasets and those entities in the selected datasets that can be linked with entities in other datasets.

1) Datasets

Dataset 1 - BiciMAD stations and status

File: bikestationbicimad_csv.csv

Req Statement Type Status Status Discussion
R1 Data belong to a smart city domain. Mandatory Satisfied Urban mobility / transport is a smart city domain.
R2 Data are available as CSV. Mandatory Satisfied Provided as CSV. Available in: BiciMad Database.
R3 Open license allows publication. Mandatory Satisfied Open reuse permitted. Use conditions available in: BiciMad dataset use conditions.
R4 Data can be easily linked with real-world entities. Mandatory Satisfied Linkable via station IDs, station address, district code, district suburb, coordinates.
R5 Documentation exists for the data. Optional Partial Basic field and dataset content descriptions available. Available in: BiciMad dataset documentation.
R6 Data may come from multiple sources. Optional Not satisfied Dirección de Tecnología e Innovación de EMT MADRID is the only publisher.
