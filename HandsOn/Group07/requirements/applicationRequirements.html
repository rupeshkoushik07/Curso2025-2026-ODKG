Proposed application: BiciMAD Linked Data App

Lightweight web application to explore BiciMAD stations from the dataset bikestationbicimad_csv.csv, enriched as Linked Data. It allows searching stations, viewing details, filtering by location, and linking CSV entities (districts/neighborhoods, streets, coordinates) with external resources (Wikidata/GeoNames/esDBpedia).

1) Scope and objectives

  • Discover nearby stations and their operational status.
  • Consult station record: address, neighborhood/district, coordinates, total slots.
  • Filter by district, neighborhood, and status.
  • Link with external entities to improve interoperability.
  • Export results to RDF and enable SPARQL queries over the generated graph.

2) Functional Requirements

IDDescriptionValidation
F1Search stations by text (name, address, coordinates).Query returns >= 1 item when a match exists.
F2Filter by district/neighborhood and status.Filters are composable and results counter shown.
F3Station detail page with metadata and outbound links (sameAs / locatedIn).External URIs are clickable and show prefixes.
F4Export to RDF (Turtle) of current results.Downloadable files include visible fields + URIs.
F5Local SPARQL endpoint over the generated graph.3 example queries executable.
F6“Map” view with markers and status legend.Markers visible.

3) Non-functional Requirements

  • Accessibility: AA contrast, keyboard navigation (Tab, Enter), alternative texts.
  • License: respect the dataset license and propagate it in exports.
  • Internationalization: interface in EN; support basic i18n (externalized strings).

Screen description

This screen displays a list of BiciMAD stations. The left panel provides the option to change the results view between filters and map. It also provides the option to choose multiple composable filters (address,free bases, etc). The right panel shows the resulting stations with name, address and current status once the filtering is made.

User interactions

  • Apply one or more filters (combinable) and see the result counter update.
  • Scroll through the results list.
  • Click a station to open its detailed record (see 5.3 Station record).
  • Use “Map” to navigate to the map view (see 5.2 Map).
